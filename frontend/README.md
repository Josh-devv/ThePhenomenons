# WellSync

Welcome to WellSync! This is a health AI companion application.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

Clone the repository and install dependencies:

```bash
npm install
```

### Development Server

Run the local development server:

```bash
npm run dev
```

### Build

To build the application for production:

```bash
npm run build
```

## Technologies Used

- React
- Vite
- TypeScript
- Tailwind CSS
